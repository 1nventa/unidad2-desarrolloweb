# unidad2-desarrolloweb

-Cómo abrir el proyecto
Requisitos previos

Antes de abrir el proyecto asegúrate de tener instalado:

 Java JDK 21 o superior
 Maven (incluido en IntelliJ IDEA)
 IntelliJ IDEA Community o Ultimate
 Git (opcional para clonar el repositorio)

 -Abrir el proyecto en IntelliJ IDEA
 Opción 1: Clonar desde GitHub

Abrir IntelliJ IDEA

Click en Get from VCS

Pegar el enlace del repositorio:

https://github.com/********/*********.git


Click en Clone

Esperar a que el IDE importe Maven automáticamente

Ejecutar el proyecto desde:

HexagonalEnfermedadApplication.java



 Opción 2: Abrir desde carpeta descargada

Descargar el ZIP del repositorio

Extraer la carpeta

Abrir IntelliJ IDEA

Seleccionar Open

Elegir la carpeta del proyecto

Esperar la sincronización de Maven


una vez abierto run : src/main/java/com/jcaa/hexagonal/HexagonalEnfermedadApplication.java
